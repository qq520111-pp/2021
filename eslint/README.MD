## 配置eslint步骤

> 根据对应的文件名替换即可。

### Hbuilderx

1. 安装插件 **eslint-js **、**eslint-plugin-vue**

2. 工具 => 设置 => 插件配置，开启自动修改和实时校验

   需要更改相关配置，点开.eslintrc.js进行配置，只需要配置rules即可。

### vscode

1. 安装eslint拓展插件，弹窗选择允许（allow）
2. 把相关.eslintrc.js文件导进去，安装对应依赖
3. 底部状态栏启动eslint，出现两个勾就行了
